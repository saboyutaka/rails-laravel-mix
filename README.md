## 開発環境構築

### ツール
* Docker, docker-compose

#### アプリケーションの構築
```
# 初回
make build

# rails s を起動
make serve
```

http://localhost:3000 にアクセス

## 技術スタック
* Docker
* docker-compose
* ruby:2.5.1
* Rails 5.2.0
* postgresql:10
* redis:4
* node:10
* yarn
* laravel-mix 4.0
